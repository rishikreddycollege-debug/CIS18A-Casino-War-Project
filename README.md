# CIS-18A Casino War Card Game (Java)

## Project Overview
This project is a Java-based implementation of the Casino War card game, developed for the CIS-18A course. The program allows a player to wager against a dealer using standard War game rules. The project demonstrates object-oriented programming concepts and fulfills all technical requirements outlined in the course rubric.

## Features
- Console-based Casino War card game
- Player wagering system with balance tracking
- Card deck creation, shuffling, and dealing
- Menu-driven gameplay using switch statements
- Object-oriented design with multiple classes
- Use of interfaces, inheritance, arrays, and control structures

## Technologies Used
- Java
- GitHub for version control

## How to Run
1. Download or clone the repository
2. Compile the Java files:
   javac *.java
3. Run the program:
   java CasinoWarGame

## Project Structure
- CasinoWarGame.java – Main game controller
- Deck.java – Deck and shuffle logic
- Card.java – Card representation
- Player.java – Player balance and actions
- Playable.java – Interface for gameplay
- Shufflable.java – Interface for shuffle behavior
